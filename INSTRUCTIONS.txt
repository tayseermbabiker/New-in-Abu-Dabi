## TECHNICAL SEO TASKS

### 8. Implement Schema Markup

**Add structured data to all pages:**

#### LocalBusiness Schema
For every business listing, add:
```json
{
  "@context": "https://schema.org",
  "@type": "LocalBusiness",
  "name": "[Business Name]",
  "image": "[Business Image URL]",
  "address": {
    "@type": "PostalAddress",
    "streetAddress": "[Full Address]",
    "addressLocality": "Abu Dhabi",
    "addressRegion": "Abu Dhabi",
    "addressCountry": "AE"
  },
  "geo": {
    "@type": "GeoCoordinates",
    "latitude": "[Latitude]",
    "longitude": "[Longitude]"
  },
  "telephone": "[Phone Number]",
  "priceRange": "[$ or $$ or $$$]",
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "[Google Rating]",
    "reviewCount": "[Number of Reviews]"
  }
}
```

#### FAQPage Schema
For all FAQ sections:
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "[Question text]",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "[Answer text]"
    }
  }]
}
```

#### BreadcrumbList Schema
For navigation:
```json
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [{
    "@type": "ListItem",
    "position": 1,
    "name": "Home",
    "item": "https://newinabudhabi.com/"
  }, {
    "@type": "ListItem",
    "position": 2,
    "name": "[Category Name]",
    "item": "[Category URL]"
  }]
}
```

#### CollectionPage Schema
For directory pages:
```json
{
  "@context": "https://schema.org",
  "@type": "CollectionPage",
  "name": "[Page Title]",
  "description": "[Page Description]",
  "url": "[Page URL]"
}
```
